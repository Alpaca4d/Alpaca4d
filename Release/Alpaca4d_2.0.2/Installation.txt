*************** Alpaca4d installation ***************
- Download Alpaca4d latest release from Food4Rhino: www.food4rhino.com/app/Alpaca4d 
- Unzip the file
- Copy the folder "Alpaca4d" in the Grasshopper Libraries folder (File -> Special Folder -> Components)
- Copy the CONTENT of the UserObjects folder into Grasshopper user objects folder (File -> Special folders -> User Objects)
- Restart Rhino
